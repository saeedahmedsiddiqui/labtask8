List in HTML

Order List

An ordered list displays items in a specific sequence, using numbers by default.

For Example

  1. Coding
  2. Playing Games
  3. Read a Book

Unorder List

An unordered list displays items without a specific order, using bullets by default.

For Example

  • Coding
  • Playing Games
  • Read a Book
